# 📊 메르 주간보고 요구사항

## 🎯 목적
메르의 블로그 포스트를 주간 단위로 분석하여 투자 인사이트를 도출하는 AI 기반 주간보고 시스템

## ⚡ 핵심 원칙
- **AI 직접 분석**: 키워드 매칭이나 외부 API 사용 금지, Claude가 직접 내용을 이해하고 분석
- **실제 데이터 기반**: 더미 데이터나 랜덤 값 사용 금지, 실제 블로그 포스트 분석
- **투자 관점**: 모든 분석은 투자자 관점에서 유용한 인사이트 도출
- **전문적 보고서**: 임원진이 읽을 수 있는 수준의 전문적이고 명확한 보고서

## 📋 주간보고 구성 요소

### 1. 주간 요약 (Executive Summary)
- **주간 하이라이트**: 가장 중요한 3-5개 인사이트
- **시장 전망**: 향후 1-2주 시장 전망과 투자 전략
- **리스크 요인**: 주의해야 할 리스크와 대응 방안
- **기회 요인**: 포착 가능한 투자 기회

### 2. 포스트별 분석
각 포스트에 대해 Claude가 직접 분석:
- **핵심 메시지**: 포스트의 핵심 내용 1-2문장 요약
- **투자 시그널**: 긍정/부정/중립 및 그 근거
- **영향 받는 종목/섹터**: 관련 종목이나 산업 섹터
- **시장 영향도**: 높음/중간/낮음 (근거 포함)

### 3. 카테고리별 트렌드
Claude가 포스트 내용을 읽고 자동 분류:
- **글로벌 매크로**: 세계 경제, 중앙은행 정책, 지정학적 이슈
- **국내 시장**: 한국 경제, KOSPI/KOSDAQ, 국내 정책
- **섹터 분석**: IT, 바이오, 제조업 등 산업별 동향
- **개별 종목**: 특정 기업에 대한 분석
- **투자 전략**: 포트폴리오, 자산배분, 리스크 관리

### 4. 종목 언급 분석
- **주요 언급 종목**: 빈도수가 아닌 중요도 기준 정렬
- **감정 분석**: Claude의 문맥 이해 기반 감정 분석
- **투자 의견**: 각 종목에 대한 메르의 관점 요약
- **액션 아이템**: 매수/매도/관망 시그널

### 5. AI 인사이트
Claude가 생성하는 고급 분석:
- **패턴 인식**: 반복되는 테마나 패턴
- **모순점 발견**: 이전 주와 상충되는 관점
- **숨겨진 연결고리**: 겉으로 드러나지 않은 관련성
- **예측 분석**: 향후 전개 시나리오

## 🤖 Claude 분석 프로세스

### 1단계: 포스트 읽기
```javascript
// 실제 포스트 내용을 Claude가 읽고 이해
const analysis = await claude.analyze({
  content: post.content,
  context: "투자 블로그 포스트 분석",
  task: "핵심 투자 인사이트 추출"
});
```

### 2단계: 맥락 이해
- 이전 포스트와의 연관성 파악
- 시장 상황 고려
- 메르의 투자 스타일 반영

### 3단계: 인사이트 도출
- 단순 요약이 아닌 투자 관점의 해석
- 실행 가능한 투자 전략 제시
- 리스크와 기회 균형있게 제시

### 4단계: 보고서 작성
- 임원진이 읽기 쉬운 구조
- 핵심 내용 우선 배치
- 시각적 요소 활용 (차트, 표, 아이콘)

## 📊 UI/UX 요구사항

### 메인 대시보드
- **주간 스코어카드**: 핵심 지표 한눈에 보기
- **트렌드 차트**: 감정 변화, 종목 언급 추이
- **액션 리스트**: 즉시 실행 가능한 투자 아이디어
- **리스크 모니터**: 주의 사항 하이라이트

### 상세 보고서
- **인터랙티브 차트**: 클릭하여 상세 정보 확인
- **드릴다운**: 요약 → 상세 → 원문 단계적 접근
- **검색 기능**: 특정 종목이나 키워드 빠른 검색
- **내보내기**: PDF, Excel 형식 지원

## 🔧 기술 구현

### Claude 통합
```typescript
interface WeeklyReportAnalyzer {
  // 포스트 직접 분석 (키워드 매칭 X)
  analyzePost(content: string): Promise<PostAnalysis>;
  
  // 투자 시그널 도출 (외부 API X)
  extractInvestmentSignals(analysis: PostAnalysis): InvestmentSignal[];
  
  // 종합 보고서 생성
  generateReport(posts: BlogPost[]): Promise<WeeklyReport>;
}
```

### 데이터 구조
```typescript
interface PostAnalysis {
  summary: string;           // 핵심 요약
  sentiment: {
    score: number;          // -1 ~ 1
    reasoning: string;      // Claude의 판단 근거
  };
  impactedStocks: {
    ticker: string;
    impact: 'positive' | 'negative' | 'neutral';
    confidence: number;     // 0 ~ 1
    reasoning: string;
  }[];
  category: string;          // Claude가 판단한 카테고리
  marketImpact: 'high' | 'medium' | 'low';
  keyInsights: string[];     // 핵심 인사이트 리스트
}

interface WeeklyReport {
  period: {
    start: Date;
    end: Date;
  };
  executiveSummary: string;
  marketOutlook: string;
  risks: Risk[];
  opportunities: Opportunity[];
  postAnalyses: PostAnalysis[];
  stockMentions: StockMention[];
  aiInsights: AIInsight[];
  actionItems: ActionItem[];
}
```

## 🧪 품질 기준

### 분석 품질
- **정확도**: 투자 시그널 정확도 > 80%
- **일관성**: 동일 포스트 재분석시 유사한 결과
- **깊이**: 표면적 분석이 아닌 심층 분석
- **실용성**: 실제 투자 결정에 도움되는 인사이트

### 보고서 품질
- **가독성**: 5분 이내 핵심 파악 가능
- **전문성**: 금융권 보고서 수준
- **시각화**: 복잡한 데이터를 쉽게 이해
- **행동지향**: 구체적인 액션 아이템 제시
